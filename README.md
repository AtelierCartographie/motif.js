# Motif

Un projet TypeScript avec Vite et pnpm.

## Installation

```bash
pnpm install
```

## Développement

```bash
pnpm run dev
```

## Build

```bash
pnpm run build
```

## Exemple avec Observable Plot

Pour ajouter Observable Plot à votre projet:

```bash
pnpm add @observablehq/plot
```

## Structure du projet

```
motif.js/
├── src/
│   ├── main.ts
│   └── style.css
├── public/
├── index.html
├── vite.config.ts
├── tsconfig.json
```
